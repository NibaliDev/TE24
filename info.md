[[fill]align][sign][#][0][width][,][.precision][type]
Alla delar är valfria, och betydelserna är:

Del	Exempel	Betydelse
fill	1, _, 0	Tecknet som används för utfyllnad
align	<, >, ^, =	Justering
sign	+, -, ' '	Hur positiva/negativa tal visas
#	#	Alternativ form (t.ex. 0x1f istället för 1f)
0	0	Nollutfyllnad (gammal stil, motsvarar fill='0', align='=')
width	10	Fältbredd
,	,	Tusenseparator
precision	.2	Antal decimaler eller längd för strängar
type	f, d, x, e, s …	Typ/format
